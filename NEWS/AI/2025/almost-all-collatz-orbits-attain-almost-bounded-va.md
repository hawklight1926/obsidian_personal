# Almost all Collatz orbits attain almost bounded values

## URL
https://mathvideos.org/2023/terence-tao-almost-all-collatz-orbits-attain-almost-bounded-values/

## 要約
テレンス・タオがコラッツ予想に関する画期的な研究成果を発表しました。彼は「ほとんど全てのコラッツ軌道は、ほとんど有界な値に到達する」ことを示し、コラッツ数列が際限なく発散する可能性が非常に低いことを数学的に裏付けました。

この結果は、任意の初期値から始まる数列が最終的に1に収束するという未解決のコラッツ予想自体を解決するものではありません。しかし、その根底にある性質の理解を深める重要な一歩であり、確率論的アプローチと数論的組み合わせを用いた画期的なアプローチが特徴です。
