# Almost all Collatz orbits attain almost bounded values

## URL
https://mathvideos.org/2023/terence-tao-almost-all-collatz-orbits-attain-almost-bounded-values/

## 要約
テレンス・タオが、未解決の数学問題であるコラッツ予想に関して画期的な成果を発表しました。
彼の研究によると、ほとんど全てのコラッツ軌道は、無限に発散することなく「ほぼ有界な値」に到達することが示されました。
これは、あらゆる正の整数が最終的に1に達するというコラッツ予想の完全な証明ではありません。
しかし、その挙動の理解を深める上で非常に重要な進展であると評価されています。
