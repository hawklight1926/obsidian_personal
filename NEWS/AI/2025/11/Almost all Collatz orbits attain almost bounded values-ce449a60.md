# Almost all Collatz orbits attain almost bounded values

## URL
https://mathvideos.org/2023/terence-tao-almost-all-collatz-orbits-attain-almost-bounded-values/

## 要約
数学者テレンス・タオが、未解決のコラッツ予想（3n+1問題）に関して画期的な進展を示しました。彼は、コラッツ予想自体は証明していないものの、「ほとんど全てのコラッツ軌道が、ほとんど有界な値に達する」ことを証明。これは、軌道の最大値が初期値に対して極めて緩やかにしか増加しないことを意味します。この成果は、予想の構造理解を大きく深める重要な一歩であり、今後の研究に大きな影響を与えると期待されています。
