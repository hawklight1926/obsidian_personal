# Claude Code Is Down

## URL
https://status.claude.com/incidents/538r2y9cjmhk

## 要約
2024年3月6日、太平洋時間午前11時25分頃から午後12時45分にかけて、Claude.aiのウェブインターフェースで部分的なサービス停止が発生しました。

この影響で、多くのユーザーがClaude.aiへのアクセスやメッセージの送信ができない状態に陥りました。

ただし、Claude APIやモバイルアプリなど、他のClaudeサービスは影響を受けずに稼働していました。

午後12時45分には問題が解決され、現在、全てのサービスは正常に機能しています。
