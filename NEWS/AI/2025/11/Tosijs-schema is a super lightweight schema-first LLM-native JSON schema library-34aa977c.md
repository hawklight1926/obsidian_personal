# Tosijs-schema is a super lightweight schema-first LLM-native JSON schema library

## URL
https://www.npmjs.com/package/tosijs-schema

## 要約
`tosijs-schema`は、TosiJSプロジェクトのデータ構造を定義するためのnpmパッケージです。JSON Schema形式でスキーマ定義を提供し、それに基づいてTypeScriptの型定義を自動生成します。これにより、データの一貫性を保ち、バリデーション、コード生成、IDEサポートを可能にします。開発効率の向上とエラーの早期発見に貢献する、TosiJSに特化した基盤ツールです。
