## 要約
Gemini APIに新たに統合されたFile Search Toolは、完全管理型のRAGシステムとして、データに基づいた正確な回答を実現します。
・開発者は自前の検索パイプラインを構築する必要がなく、データ連携が容易になります
・クエリ時のストレージと埋め込み生成は無料で、初期インデックス作成のみ1百万トークンあたり0.15ドルです
・Google AI Studioのデモアプリで実際の動作を確認でき、サポートボットや社内ナレッジ支援などへの応用が進んでいます
