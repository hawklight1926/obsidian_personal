## 要約
Gemini APIに新たに追加されたFile Search Toolは、完全管理型のRAGシステムとして、データに基づいた正確な回答を実現します。
・開発者は複雑な検索パイプラインを管理せず、データに基づいた応答生成に集中できます
・ストレージとクエリ時の埋め込み生成は無料で、初期インデックス作成のみ1ミリオントークンあたり0.15ドルの費用が発生します
・Google AI Studioのデモアプリで実際の動作を確認でき、サポートボットや社内ナレッジ助手などへの応用が進んでいます
