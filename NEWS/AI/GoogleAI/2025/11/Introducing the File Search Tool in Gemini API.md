## 要約
GoogleはGemini APIに「File Search Tool」を追加した。これは自社データを活用して高精度な回答を生成する、完全管理型のRAGシステムである。クエリ時のストレージと埋め込み生成は無料で、初期のファイル索引作成のみ低コストがかかる。開発者はドキュメントやデモアプリで今すぐ利用を開始できる。
