## 要約
GoogleはGemini APIに「File Search Tool」を追加した。これは自社データを活用してGeminiの回答精度を高める、完全管理型のRAGシステムである。クエリ時のストレージと埋め込み生成は無料で、初期のファイル索引作成のみ低コストがかかる。開発者は、より正確で検証可能な回答を実現するアプリケーションを簡単に構築できる。
