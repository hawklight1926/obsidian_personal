## 要約
Wailsは、Go言語とWeb技術（HTML/JS/CSS）を用いてデスクトップアプリケーションを開発するためのフレームワークです。
GoコードとWebフロントエンドを単一のバイナリとしてバンドルすることを可能にし、プロジェクトの作成からコンパイル、バンドルまでを容易にするツールを提供します。
軽量なアプリケーションの構築を目指すGoプログラマーを主なターゲットとし、メニューやダイアログなどのネイティブ要素もサポート。
サーバーを立てずにWeb UIを組み込みたい場合や、Electronのより軽量な代替を探している場合に有用です。
