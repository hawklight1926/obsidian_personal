## 要約
Gemini APIに、完全管理型のRAGシステム「File Search Tool」が追加されました。
・ユーザーデータを組み合わせることで、より正確で検証可能な応答を実現します。
・ストレージと検索時の埋め込み生成は無料で、初期インデックス作成のみ課金されます。
・開発者は複雑なインフラ管理から解放され、アプリケーション開発に集中できます。
