## 要約
マイクロソフトは「Microsoft Agent 365」を発表し、AIエージェントの一括管理を可能にします。
- 企業内の全てのAIエージェントを可視化し、アクセス制御やセキュリティ監視を行います。
- エージェントの行動を監査し、組織への影響を可視化することで、安全な運用を実現します。
- Microsoft 365管理センターから利用でき、DefenderやPurviewなどの既存サービスと連携します。
