## 要約
Microsoftは「Microsoft Agent 365」を発表し、AIエージェントの一元的な管理を可能にします。
- あらゆるAIエージェントの完全なインベントリを取得し、可視化します。
- 最小権限のアクセス制御や監査ログにより、セキュリティとガバナンスを強化します。
- エージェントの行動をプロファイリングし、組織への影響を可視化します。
- Microsoft 365管理センターから利用でき、既存のセキュリティツールと連携します。
