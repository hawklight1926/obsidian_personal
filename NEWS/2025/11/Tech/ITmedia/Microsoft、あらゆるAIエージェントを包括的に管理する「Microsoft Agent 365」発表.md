## 要約
マイクロソフトは「Microsoft Agent 365」を発表し、AIエージェントの一元的な管理を可能にします。
- 多様なAIエージェントを一覧で把握し、必要なシステムへのアクセス権のみを付与できます。
- エージェントの行動を監視して影響を可視化し、人間との協働を支援する機能も備えます。
- セキュリティやコンプライアンスを確保し、危険な行動から保護する仕組みを提供します。
Microsoft 365管理センターから利用でき、既存のセキュリティ製品などと連携します。
