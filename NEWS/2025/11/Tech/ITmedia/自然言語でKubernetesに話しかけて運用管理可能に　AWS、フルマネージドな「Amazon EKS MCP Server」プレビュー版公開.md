## 要約
AWSが、自然言語でKubernetesの操作・管理を可能にする新サービス「Amazon EKS MCP Server」の提供を開始しました。
- 従来のkubectlコマンドの知識がなくても、AIを活用してクラスターの作成や管理を対話的に行えます。
- 具体的には、クラスター作成やワークロード管理、設定ファイル生成、セキュリティ診断などの機能を備えています。
- これにより、運用負荷の軽減と、専門知識を持たないユーザーでもインフラ操作が可能になります。
