## 要約
AWSは、自然言語でKubernetesの操作・管理を可能にする新サービス「Amazon EKS MCP Server」の提供を開始しました。
- 従来のkubectlコマンドの知識がなくても、Amazon EKSクラスターの作成・管理が可能です。
- 自然言語による指示で、Kubernetesリソースの管理やアプリケーションのデプロイ用マニフェスト生成などが行えます。
- AWSのベストプラクティスに基づく推奨設定を提供し、Amazon Qとの連携による問題診断もサポートします。
