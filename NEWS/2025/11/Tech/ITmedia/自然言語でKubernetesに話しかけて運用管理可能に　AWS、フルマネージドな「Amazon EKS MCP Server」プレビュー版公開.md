## 要約
AWSは、自然言語でKubernetesの操作・管理を可能にする新サービス「Amazon EKS MCP Server」の提供を開始しました。
・kubectlコマンドの知識がなくても、Amazon EKSクラスターの作成・管理やワークロードのデプロイが行えます。
・AIによるコード生成や、問題診断と解決手順の提示などの機能を備えています。
・これにより、開発者はインフラの専門知識に依存せず、Kubernetesをより容易に利用できるようになります。
