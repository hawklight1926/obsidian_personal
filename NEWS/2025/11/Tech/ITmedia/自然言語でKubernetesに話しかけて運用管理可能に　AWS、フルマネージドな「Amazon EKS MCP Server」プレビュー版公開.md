## 要約
AWSが、自然言語でKubernetesの操作・管理を可能にする新サービス「Amazon EKS MCP Server」の提供を開始しました。
- 従来の`kubectl`コマンドの知識がなくても、AIが自然言語による指示をKubernetes操作に変換します。
- 主な機能として、EKSクラスターの作成・管理や、Kubernetesリソースの管理、アプリケーションのデプロイ用マニフェスト生成などがあります。
- これにより、運用担当者は専門知識に依存せず、より効率的なクラウドネイティブ環境の運用が可能となります。
