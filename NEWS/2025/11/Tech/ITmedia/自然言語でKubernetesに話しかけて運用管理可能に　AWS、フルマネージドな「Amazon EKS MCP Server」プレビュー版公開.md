## 要約
AWSは、自然言語でKubernetesの操作・管理を可能にする新サービス「Amazon EKS MCP Server」の提供を開始しました。
- 従来のkubectlコマンドの知識がなくても、Amazon EKSクラスターの作成や管理を会話形式で行えます。
- Kubernetesマニフェストの生成や、AWSが管理するクラスターの運用を支援する推奨アクションの提供などの機能を備えています。
- これにより、AIを活用した開発者がインフラ操作をより直感的に行える環境を実現します。
