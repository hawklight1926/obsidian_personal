## 要約
Rubrikは、AIエージェントの誤動作や不正利用によるリスクに対応する新サービス「Rubrik Agent Cloud」を発表しました。
- 本サービスは、クラウド上のAIエージェントを可視化・監視し、ポリシー逸脱を検知します。
- 問題発生時には、Rubrik Security Cloudと連携して、AIエージェントのアクションを任意の時点に巻き戻し（リワインド）できます。
- これにより、AIエージェントの暴走によるデータ破壊などの新たな事業継続リスクへの対策を提供します。
