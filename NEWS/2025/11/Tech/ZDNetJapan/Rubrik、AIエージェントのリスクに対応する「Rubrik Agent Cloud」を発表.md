## 要約
RubrikはAIエージェントの誤動作や不正利用によるリスクに対応する「Rubrik Agent Cloud」を発表しました。  
・AIエージェントの動作を可視化・監視し、ポリシー逸脱を検知します。  
・問題発生時はRubrik Security Cloudと連携し、アクションを任意の時点に巻き戻して復旧を可能にします。  
・サービスは買収したPredibaseを基盤とし、データ、ID、アプリケーションを一元管理します。
