## 要約
AWSはAIがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、リソース管理や操作を自動化できます。  
・具体的には、5000以上のAWS API呼び出しや、最新の技術情報への参照が可能です。  
・これにより、AIエージェントが標準手順に沿ったAWS操作や学習データにない新APIの利用などを実行できます。
