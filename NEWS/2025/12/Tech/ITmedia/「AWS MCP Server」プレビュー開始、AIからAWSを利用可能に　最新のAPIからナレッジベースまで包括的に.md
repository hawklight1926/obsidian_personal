## 要約
AWSはAIがAWSを利用するための新サービス「AWS MCP Server」の提供を開始しました。
・AIがAWS APIを呼び出したり、ドキュメントにアクセスしたりすることを可能にします。
・具体的には、標準的な操作手順の取得や、多数のAPI呼び出しの管理・確認などの機能を提供します。
・これにより、AIエージェントや生成AIがAWSのリソースをより効果的に活用できる環境を整えます。
