## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、AWSリソースを操作・参照できるようになります。  
・具体的には、5000以上のAWS API呼び出しや、技術ドキュメント、ブログなどの情報参照が可能です。  
・これにより、AIによるAWSの自動運用や、最新情報に基づいた適切な操作が実現されます。
