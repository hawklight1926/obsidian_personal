## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。
・AIがAWSのAPIやドキュメントを利用できるようにするための接続規格「MCP」に対応しています。
・具体的には、5000以上のAWS API呼び出しや、ナレッジベース、APIリファレンスへのアクセスを可能にします。
・これにより、AIによるAWSリソースの管理や、最新情報に基づいた操作の自動化が促進されます。
