## 要約
AWSはAIがAWSを利用するための新サービス「AWS MCP Server」の提供を開始しました。  
・AIアシスタントがAWS APIやドキュメントにアクセスし、操作を実行できるようになります。  
・具体的には、標準的な手順書の参照や、多数のAWS APIの呼び出し、最新情報の取得が可能です。  
・これにより、AIを活用したAWSリソースの管理や問い合わせ効率が向上することが期待されます。
