## 要約
AWSはAIがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。
- AIがAWS APIやドキュメントにアクセスし、操作や情報参照を可能にします。
- 具体的には、標準手順書の参照や、多数のAWS API呼び出し、最新API情報の取得などが行えます。
- これにより、AIエージェントがAWSリソースを自律的に管理・活用できる環境が整います。
