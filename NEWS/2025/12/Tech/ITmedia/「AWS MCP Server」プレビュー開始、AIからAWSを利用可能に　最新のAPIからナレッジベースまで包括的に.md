## 要約
AWSがAIエージェントがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。  
・最新のAWS APIからドキュメントまでを統合的にAIが利用可能になります。  
・具体的には、標準的な操作手順の取得や、多数のAPI呼び出しの最適化などが可能です。  
・AWSの技術文書やAPI仕様、ブログなどの情報への参照も行えます。
