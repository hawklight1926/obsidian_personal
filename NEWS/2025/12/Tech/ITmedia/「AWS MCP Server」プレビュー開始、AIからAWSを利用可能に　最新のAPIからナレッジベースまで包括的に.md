## 要約
AWSがAIエージェント向け新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、AWSリソースを操作・参照できるようになります。  
・具体的には、標準手順書の取得、数千を超えるAWS APIの呼び出し、最新情報の入手などが可能です。  
・これにより、AIを活用したAWSの利用と管理が効率化されます。
