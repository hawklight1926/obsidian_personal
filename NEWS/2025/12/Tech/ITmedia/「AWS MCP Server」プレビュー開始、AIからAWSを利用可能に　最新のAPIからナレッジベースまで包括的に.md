## 要約
AWSはAIがAWSを利用するための新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、操作や情報参照を可能にします。  
・具体的には、標準的な作業手順の取得や、多数のAPI呼び出し、最新情報の参照などの機能を提供します。  
・これにより、AIエージェントがAWSリソースを自律的に利用・管理できる環境が整備されます。
