## 要約
AWSは、AIがAWSを利用するための新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、操作や情報取得を可能にします。  
・具体的には、標準的な手順書の提供や、多数のAPI呼び出しの管理、最新情報の参照などの機能を備えています。  
・これにより、AIアシスタントやエージェントがAWSリソースをより効果的に活用できるようになります。
