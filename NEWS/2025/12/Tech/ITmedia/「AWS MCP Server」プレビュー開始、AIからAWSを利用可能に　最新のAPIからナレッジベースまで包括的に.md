## 要約
AWSはAIがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIやドキュメントにアクセスし、操作や情報参照を可能にします。  
・具体的には、標準手順書の取得や、多数のAPI呼び出し、最新情報の学習などが行えます。  
・これにより、AIエージェントがAWSリソースを自律的に管理・活用できる環境を提供します。
