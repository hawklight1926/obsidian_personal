## 要約
AWSがAIエージェントがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIを呼び出したり、ドキュメントにアクセスしたりすることを可能にします。  
・具体的には、標準的な操作手順の取得や、多数のAPI呼び出しの管理・最適化などの機能を提供します。  
・これにより、AIはAWSの最新情報や関連リソースを参照しながら、効果的な操作を行えるようになります。
