## 要約
AWSがAIエージェントがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。
- 最新のAWS APIからドキュメントまでを統合的にAIが利用可能になります。
- 具体的には、標準手順書の参照や、多数のAPI呼び出し、関連情報の検索などの機能を提供します。
- これにより、AIがAWSのリソースやサービスを自律的に操作・学習することが可能となります。
