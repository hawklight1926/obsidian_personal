## 要約
AWSがAIによる自社サービスの利用を可能にする「AWS MCP Server」の提供を開始しました。
・AIがAWS APIやドキュメントにアクセスし、操作や情報参照を実行できます。
・具体的には、標準的な作業手順の取得や、多数のAPI呼び出し、最新情報の参照などの機能を備えています。
・これにより、生成AIやAIエージェントがAWSをより自律的に利用できる環境が整います。
