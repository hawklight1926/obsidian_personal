## 要約
AWSは、AIがAWSを利用できるようにする新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWS APIを呼び出したり、ドキュメントにアクセスしたりすることを可能にします。  
・具体的には、標準的な操作手順の取得や、多数のAPIの呼び出し、関連情報の参照などの機能を提供します。  
・これにより、AIエージェントやAIアシスタントがAWSをより効果的に利用できる環境が整います。
