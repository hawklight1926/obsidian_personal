## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWSのAPIやドキュメントにアクセスし、操作できるようにするための接続規格（MCP）に対応しています。  
・具体的には、5000以上のAWS API呼び出しや、ナレッジベース、APIリファレンスへのアクセスを可能にします。  
・これにより、AIがAWSリソースの管理や最新情報の取得を自動化するための標準手順（SOP）を利用できるようになります。
