## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。
・AIがAWSを利用できるようにするための接続仕様「MCP」に対応するサーバーです。
・具体的には、AWSのAPI呼び出しや、ドキュメント・ナレッジベースへのアクセスを可能にします。
・これにより、AIは標準的な操作手順の取得や、多数のAWS APIの利用などが行えるようになります。
