## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。
・AIがAWSのAPIやドキュメントを利用できるようにするための接続規格「MCP」に対応しています。
・具体的には、AWS APIの呼び出しや、ナレッジベース、技術ブログなどの情報へのアクセスを可能にします。
・これにより、AIはAWSの最新サービスを含む広範なリソースを自律的に利用できるようになります。
