## 要約
AWSがAIエージェントがAWSを利用できるようにする新サービス「AWS MCP Server」の提供を開始しました。  
・MCP（Model Context Protocol）は、AIエージェントが外部ツールを利用するための標準規格です。  
・本サービスでは、AIがAWS APIの呼び出しやドキュメントへのアクセスなどを自律的に実行できます。  
・具体的には、標準作業手順の取得や、多数のAWS APIの利用、関連情報の参照などが可能です。
