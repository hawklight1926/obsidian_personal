## 要約
AWSがAIエージェントがAWSを利用できる新サービス「AWS MCP Server」の提供を開始しました。
・AIがAWS APIを呼び出したり、ドキュメントにアクセスしたりすることを可能にします。
・具体的には、標準的な操作手順の取得や、多数のAWS APIの呼び出し、最新の技術情報へのアクセスなどの機能を提供します。
・これにより、AIアシスタントやAIエージェントがAWSリソースをより自律的に利用できる環境が整えられます。
