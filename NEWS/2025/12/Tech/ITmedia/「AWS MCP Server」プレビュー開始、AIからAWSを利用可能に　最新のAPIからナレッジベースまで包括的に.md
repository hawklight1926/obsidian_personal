## 要約
AWSがAIエージェント向けの新サービス「AWS MCP Server」の提供を開始しました。  
・AIがAWSのAPIやナレッジを利用するための標準化された接続手段を提供します。  
・具体的には、AWS APIへのアクセスやドキュメント参照など、多様なAWSリソースへの連携を可能にします。  
・これにより、AIは最大1日5000件以上のAPI呼び出しを含む、効率的なAWS操作を実行できるようになります。
