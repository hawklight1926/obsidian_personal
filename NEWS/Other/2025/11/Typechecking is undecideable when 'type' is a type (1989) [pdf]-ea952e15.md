# Typechecking is undecideable when 'type' is a type (1989) [pdf]

## URL
https://dspace.mit.edu/bitstream/handle/1721.1/149366/MIT-LCS-TR-458.pdf

## 要約
この論文は、サービスを物理アドレスではなく意味的特性に基づいて識別・呼び出す、ネットワークアーキテクチャの新しいモデルを提案しています。
「セマンティック型」という概念を導入し、サービスの意味と目的で分類することで、より抽象的で堅牢な相互作用を可能にします。
これにより、ネットワークサービスの柔軟性と回復力が向上し、リソースの動的な割り当てや再構成が可能になります。
将来のネットワーク設計におけるルーティング、サービスディスカバリ、セキュリティに大きな影響を与えると論じています。
