# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
gh-slimifyは、GitHubのユーザーインターフェースを簡素化し、よりミニマルな表示を実現するためのCSSスタイルシートです。不要なサイドバー、ヘッダー、フッターといったUI要素を非表示にすることで、コードやMarkdownコンテンツに集中できるクリーンなビューを提供します。Stylusなどのブラウザ拡張機能を通じてカスタムCSSとして適用でき、GitHubの閲覧環境をすっきりとさせます。
