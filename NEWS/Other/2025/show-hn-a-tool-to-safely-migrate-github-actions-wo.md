# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
`gh-slimify`は、GitHub Actionsのストレージを管理するためのコマンドラインツールです。
指定された日数を超えた古いビルドアーティファクトを削除することで、リポジトリの容量を節約し、GitHub Actionsの無料枠の制限内での運用を支援します。
使用するには、リポジトリオーナー、リポジトリ名、および適切な権限を持つGitHubパーソナルアクセストークンを指定します。
最新のアーティファクトを保持するオプションや、実際に削除せずに確認できるドライランモードも備えています。
これにより、不要なアーティファクトの蓄積を防ぎ、効率的なGitHub Actionsの運用が可能になります。
