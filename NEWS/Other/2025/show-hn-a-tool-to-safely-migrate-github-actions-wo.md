# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
`gh-slimify`は、GitHubの通知を効率的に管理するための`gh` CLI拡張機能です。このツールは、カスタマイズ可能なルールに基づき、指定したリポジトリや通知の種類に応じて自動的に既読（アーカイブ）する機能を提供します。

通知に埋もれることを防ぎ、本当に重要な情報に集中できるように支援します。`gh-slimify`は、GitHub通知のストレスを軽減し、よりクリーンで整理されたワークフローを実現します。
