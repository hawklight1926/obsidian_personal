# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
`gh-slimify`は、GitHubアカウントを効率的に整理し、利用体験をスリム化するためのPythonスクリプトです。
このツールは、不要になったスター付きリポジトリのアンスタースルー、ウォッチしているリポジトリの監視解除、古いGistやプライベートフォークの削除といったクリーンアップ作業を自動化します。
最終更新日やトピックに基づいた柔軟なルール設定が可能で、実行前にはドライランモードで変更内容を確認できます。
増えすぎたGitHubコンテンツを整理し、必要な情報へのアクセスを向上させることが主な目的です。
