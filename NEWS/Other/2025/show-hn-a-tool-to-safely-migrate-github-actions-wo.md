# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
`gh-slimify`は、GitHub CLIの拡張機能であり、gh拡張機能のキャッシュや古いバージョンを削除するツールです。
これにより、ディスクスペースを節約し、ローカルのgh拡張機能ディレクトリを整理できます。
基本的に、各拡張機能の最新バージョンのみを残し、それ以外の古いバージョンをクリーンアップします。
また、特定のエクステンションの全バージョンを削除する機能も備わっています。
