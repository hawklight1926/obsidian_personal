# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
「gh-slimify」は、GitHub CLIの拡張機能です。GitHubのウェブUIから、探索、スポンサー、サイドバーなどの不要な要素や邪魔な部分を削除し、よりすっきりとしたシンプルな表示を実現します。

これは、`gh browse`コマンドを使ってブラウザでページを開き、カスタムのCSSとJavaScriptを注入することで実現されます。プロジェクトは、より集中しやすいGitHub体験を提供することを目的としています。
