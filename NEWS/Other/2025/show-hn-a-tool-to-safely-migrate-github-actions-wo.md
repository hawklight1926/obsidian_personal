# Show HN: A tool to safely migrate GitHub Actions workflows to Ubuntu-slim runner

## URL
https://github.com/fchimpan/gh-slimify

## 要約
「gh-slimify」は、GitHub CLIの拡張機能として提供されるシンプルなスクリプトです。
古くなった未使用のGitHub Codespacesおよび関連リソースを効率的に削除し、無駄なコストやリソース消費を削減することを目的としています。
最終使用日時を基準に削除対象を特定し、ドライランや対話モードといった安全に実行するためのオプションも備わっています。
このツールは、Codespacesの定期的なクリーンアップを自動化するのに役立ちます。
