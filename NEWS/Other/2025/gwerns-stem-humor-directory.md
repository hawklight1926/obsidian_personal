# Gwern's "Stem Humor" Directory

## URL
https://gwern.net/doc/math/humor/index

## 要約
Gwernの記事「Mathematics Humor」は、数学コミュニティで共有されるユーモアの性質を解説しています。このユーモアは、数学の専門知識、記号論理、そして概念に基づいているため、部外者には理解しにくい内輪的な性質を持つと説明しています。

記事では、定義、公理、無限、統計、他分野との対比、数学者のタイプなど、様々なジョークのカテゴリーを紹介。これらが数学コミュニティの結束を強め、数学的思考を育む役割も持つと指摘しています。
