# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
2025年11月という未来の視点から、AIブーム後の現状を考察した記事です。大規模言語モデル（LLM）は普及したものの、GPT-4以降の性能向上は鈍化し、特に期待されたAIエージェントは未だに脆く予測不能だと指摘します。

これは、LLMが真の理解、長期記憶、常識に欠け、幻覚や複雑な推論といった根本的な課題を抱えるためです。結果として完全な自律エージェントではなく、人間が介入するシステムが主流となっています。

著者は、統計的パターンマッチングを超えるAIの根本的進歩か、より堅牢なエンジニアリングパラダイムが必要だと結論付けています。
