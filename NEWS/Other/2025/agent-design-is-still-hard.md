# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
LLMベースのAIエージェント開発は、多くの根本的な困難に直面しているとこの記事は指摘します。主な課題として、複雑な状態管理、不確実な推論、デバッグの難しさ、信頼性の欠如を挙げ、既存フレームワークも問題を抽象化するだけで複雑さを増大させると述べています。著者は、エージェントを過信せず、限定的なタスクへの適用、人間の介入を前提としたシンプルな設計、そして既存のソフトウェア工学の原則に従うことの重要性を訴えています。
