# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
筆者はAIエージェントの可能性に期待していたものの、Auto-GPTなどの初期の試みは実用性に乏しく、実現の難しさを痛感していると述べます。現在のLLMは次に来るトークンを予測するモデルに過ぎず、真のプランニングやデバッグ能力がないため、複数ステップのタスクを自律的かつ信頼性高く実行させるのは極めて困難です。

環境全体に渡るエージェントの複雑なデバッグの難しさも相まって、現状のAIエージェントは期待される汎用AIとは大きく乖離しており、根本的に新しいアプローチが必要だと筆者は指摘しています。
