# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
記事「Agents Are Hard」は、LangChainなどの既存エージェントフレームワークが現実世界の複雑性に対応できていないと指摘します。現実世界では行動結果が不確かで環境が常に変化するため、エージェントはネットワークエラーやウェブサイトの変更といった予期せぬ障害に直面しがちです。

著者は、エージェントを制御する「モデル」自体が、現実世界の不確実性やフィードバックループを考慮して設計されるべきだと主張。現状の決定木のような単純なアプローチでは根本的に不十分であり、より根源的なレベルでの再考が必要であると提言しています。
