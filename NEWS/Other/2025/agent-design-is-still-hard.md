# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
本記事は、大規模言語モデル（LLM）を用いたAIエージェントの構築が極めて難しい現状を解説するものです。エージェントは、LLMのハルシネーションやツールの誤用、ステートレス性、コンテキストの制限、高コスト、非決定性といった課題に直面しています。

そのデバッグや評価は困難を極め、既存のソフトウェア開発プラクティスとの統合も未解決の課題です。筆者は、エージェントは魔法ではなく、根本的な問題解決が必要であり、信頼性のある実用化にはまだ距離があると指摘しています。
