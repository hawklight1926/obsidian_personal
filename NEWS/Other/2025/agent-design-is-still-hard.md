# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
この記事は、LLMエージェントが過大評価されており、本質的な限界を持つため、うまく機能しないと主張しています。エージェントは、真の理解力、記憶力、そして曖昧なタスクへの対応能力が欠如しているため、失敗に終わることが多いと指摘。

筆者は、人間のような本能や世界モデルを持たないエージェントに自律性を求めるのではなく、人間が主導するインタラクティブなシステムにおいて、LLMを強力なツールとして活用する方向へ進むべきだと提言しています。
