# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
この記事は、LLMエージェントが謳われるほど単純ではなく、多くの「エージェント」が実質的に高度なプロンプトエンジニアリングの延長に過ぎないと指摘しています。真の自律性、長期的な計画、記憶、環境との相互作用といった本質的な課題は、LLM単体では解決できません。筆者は、環境をモデル化し、計画し、学習する能力を持つ、根本的に異なるアプローチが必要だとし、より優れたAIエージェントの実現にはブレークスルーが不可欠だと結論付けています。
