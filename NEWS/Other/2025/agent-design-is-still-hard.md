# Agent design is still hard

## URL
https://lucumr.pocoo.org/2025/11/21/agents-are-hard/

## 要約
この記事は、AIエージェント、特にLLMベースのエージェントの構築は、一般に考えられているよりもはるかに難しいと指摘しています。
LLMは本質的にステートレスであり、長期的な記憶、文脈、アイデンティティを維持することに根本的な課題を抱えています。
ツールを使えても、その背後にあるシステムのセマンティクスや状態を真に理解しているわけではありません。
現在の「エージェント」は、より高度なスクリプトやオーケストレーション層に近いものであり、真の自律的な知能にはまだ大きな隔たりがあると筆者は述べています。
誇大広告に惑わされず、エージェントの実現には複雑な課題が山積していることを強調しています。
