# Unicode Binary Input Terminal

## URL
https://hackaday.io/project/192644-unicode-binary-input-terminal

## 要約
「Unicodeバイナリ入力ターミナル」プロジェクトは、従来のキーボードとは異なる、文字入力の新しいアプローチを提案しています。
これは、8つのビット入力キーと1つの確定キーを使用し、文字のバイナリ表現を直接入力することで、任意のUnicode文字を生成するデバイスです。
キーボードレイアウトに縛られず、あらゆる文字を直接入力できる点が特徴です。
主に、任意のUTF-8シーケンスの入力や、文字エンコーディングの仕組みを学ぶための教育ツールとしての活用が想定されています。
