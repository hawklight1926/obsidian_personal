# Unicode Binary Input Terminal

## URL
https://hackaday.io/project/192644-unicode-binary-input-terminal

## 要約
このプロジェクトは、最小限の物理キーで任意のUnicode文字を入力するための「Unicode Binary Input Terminal」を構築するものです。

8つのビット入力キーと1つのコミットキーを使い、各文字をUTF-8バイト列として直接入力します。これは、従来のキーボードでは困難だった多言語や特殊文字の効率的な入力を可能にする目的があります。

Raspberry Pi Picoを基盤とし、物理的なボタンとLCDで構成されており、直感的な入力方法を提供します。
